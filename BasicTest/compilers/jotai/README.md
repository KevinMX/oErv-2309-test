# jotai

jotai 是从 AnghaBench 中抽取修改的可编译可运行的代码测试集。

```
jotai
├── log #测试日志
│   ├── anghaLeaves.fail
│   ├── anghaMath.fail
│   ├── old
│   │   ├── jotai.log
│   │   └── jotai.md
│   └── test.log.gz
├── method.md   #测试方法
├── README.md   #测试说明
├── result.md   #测试结果
└── src #测试源码
    ├── Makefile
    └── singletest.lua
    ```
