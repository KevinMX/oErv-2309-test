# AnghaBench 测试

AnghaBench 包含了一百万个可编译的 C 文件，可用于对 C 编译器进行编译测试。

## 文件树

```
AnghaBench
├── log
│   ├── AnghaBench.log.gz
│   ├── arch.md
│   ├── compilation_log.tar.gz
│   └── testcase_problem.md
├── method.md   #测试方案
├── README.md   #测试说明
├── result.md   #测试结果分析
└── src
    └── compile.sh  #测试用脚本
```
